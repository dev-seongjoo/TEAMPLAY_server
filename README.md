# TEAMPLAY_server

### 💻 팀 프로젝트가 필요한 당신을 위한 플랫폼 !

> TEAMPLAY는 팀 프로젝트가 필요한 모두를 위한 팀 빌딩 서비스입니다.   
> 현직자들의 사이드 프로젝트나 스타트업 창업 멤버를 모집하는 기존의 플랫폼과 달리 주니어 개발자를 목표로 하는 취준생들이 중심이 되는 서비스입니다.
